## рҹҺҜ When to Use Each Button

### Use MIC BUTTON рҹҺӨ when:
- вң… You want to dictate text
- вң… Need to edit before sending
- вң… Typing is inconvenient
- вң… Want precise text input
- вң… One-time voice input

### Use VOICE MODE вӯ• when:
- вң… Want hands-free conversation
- вң… Having a back-and-forth chat
- вң… Brainstorming ideas
- вң… Quick questions
- вң… Natural conversation flow
- вң… Like talking to ChatGPT/Gemini

### Use SEND BUTTON вҶ‘ when:
- вң… Typing on keyboard
- вң… Pasting text
- вң… Sending code
- вң… Precise formatting needed
- вң… Standard text chat
